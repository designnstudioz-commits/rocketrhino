# Rocket Rhino

Figma Make project - Interactive desktop application built with React and Tailwind CSS.

## Setup

1. Clone the repository:
```bash
git clone https://github.com/faisalmunir-afk/rocketrhino.git
cd rocketrhino
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

## Project Structure

- `src/app/App.tsx` - Main application component
- `src/app/components/` - React components
- `src/imports/` - Figma imported assets and components
- `src/styles/` - CSS and theme files

## Tech Stack

- React
- Tailwind CSS v4
- TypeScript
- Vite
- shadcn/ui components
